# IntelligentLamp

此 APP 可作为硬件蓝牙控制器和本地音乐播放器使用，主要具备以下功能：

1. 本地登陆及注册功能、手机本地音乐列表获取及播放功能。

2. 充当蓝牙控制器的作用，并结合色环采集颜色，进而控制灯光的颜色变化及音乐播放。
